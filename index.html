Projetos em Destaque

A Baleia Azul de Valença – Jogo RPG Narrativo

Inspirado na lenda folclórica da minha cidade natal, este jogo foi desenvolvido durante a disciplina de Programação Orientada a Objetos na PUCRS como uma forma de aplicar, de maneira criativa, os conceitos técnicos de POO. A experiência propõe uma jornada textual onde o jogador explora diferentes ambientes históricos, interage com objetos simbólicos e resolve enigmas para impedir o despertar de uma criatura mítica.

A estrutura do código foi cuidadosamente pensada para representar classes como salas, objetos, ferramentas e uma engine que controla toda a narrativa. Cada elemento foi escrito de forma modular e extensível, permitindo que o jogo seja expandido com facilidade. Mais do que um projeto acadêmico, essa aplicação representa minha capacidade de unir lógica, identidade cultural e estrutura técnica em um único produto interativo.

API RESTful – Sistema de PDV (Frente de Caixa)

Criada como projeto final do curso Back-End da Cubos Academy, esta API simula o funcionamento essencial de um sistema de frente de caixa, utilizado no dia a dia por comércios de pequeno a médio porte. Ela oferece funcionalidades completas para o gerenciamento de usuários, produtos, categorias e pedidos, com foco na segurança das rotas e integridade dos dados.

Utilizando Node.js e PostgreSQL, a aplicação segue padrões RESTful, implementando autenticação com JWT, validação de entrada e respostas com status codes padronizados. Além da lógica técnica, este projeto me permitiu explorar práticas como organização em camadas, uso de scripts SQL para estruturação do banco de dados e construção de rotas robustas, tudo pensado para refletir um cenário real de desenvolvimento back-end.
